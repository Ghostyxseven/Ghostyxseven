# 🎮 Quiz Interativo em JavaScript

Este é um jogo de **perguntas e respostas** desenvolvido com HTML, CSS e TypeScript/JavaScript. O objetivo é testar o conhecimento do usuário em diferentes categorias, com feedback instantâneo sobre as respostas.

## 💡 Funcionalidades

### Funcionalidades Básicas:
- ✅ Exibição de perguntas e múltiplas opções de resposta.
- ✅ Armazenamento das perguntas/respostas em arrays.
- ✅ Uso de **objetos** para representar perguntas.
- ✅ Estrutura com **classes** para controlar o estado do jogo.
- ✅ Utilização de **if/else**, **for**, etc., para a navegação entre perguntas e validação das respostas.

### Funcionalidades Adicionais:
- 🗂 Categorias personalizadas de perguntas (Ex: Tecnologia, História, Ciências).
- 💻 Interface simples e responsiva com HTML e CSS.
